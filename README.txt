Esse projeto é destinado à criação de uma aplicação utilizando os conceitos de arquiteturais HTTP/2 REST.

Frameworks e Tecnologias utilizadas:

BackEnd:
	-NodeJS
	-Express
		

Infraestrutura:
	- Docker
	- MongoDB

FrontEnd:
	- Vue.js V2
	- Vuetify
	

Configurações:
 - Utilizando MLab para armazenar os documentos do MongoDB.
